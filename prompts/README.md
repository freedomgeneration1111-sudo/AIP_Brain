# Grok Build Prompts

This directory contains the Grok Build prompts used to implement AIP features.
These are the actual directives given to the AI coding agent during development.

They serve as:
- Historical record of implementation decisions
- Templates for future similar implementations
- Evidence of the AI Poiesis methodology in practice
  (DEFINER directs, agent implements, DEFINER verifies)

Each prompt corresponds to one or more commits. The prompt filename
or content describes what feature was built.

For the current build roadmap see: ROADMAP.md
For architectural decisions see: docs/decisions/
